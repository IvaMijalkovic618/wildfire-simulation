# Wildfire Simulation

This project simulates the spread of a wildfire on a 2D grid using a probabilistic model.

The simulation generates a forest using a random walk algorithm and models how fire spreads between neighboring cells over time. Each cell can represent different states such as forest, burning, or burned.

## Features
- Random forest generation
- Probabilistic fire spread
- Time-based simulation (ticks)
- Configurable parameters via input file
- Real-time visualization using Java Swing

## Technologies
- Java
- Swing (GUI)
